# **Module 5 — Equation Explainer**
- Identify all equations or mathematical expressions within each section.  
- Extract the equation text exactly as written—no rewriting or symbolic modification.  
- Explain the meaning of each main variable or term  
- Connect the equation to the surrounding section’s narrative  
- **DO NOT**  Introduce new mathematics not found in the text

**Outputs**
For each section:
- A list of extracted equations  
- A short explanation (2–5 sentences) of each equation  
- Variable definitions where available
