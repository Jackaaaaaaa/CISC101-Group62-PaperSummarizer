# **Module 4 — Rendering & Refinement**

## **Purpose**
Generate the final polished Markdown output.

## **Responsibilities**
- Construct the **Paper Summary** (200 ± 50 words).  
- Build the **Section-by-Section Table**.  
- Produce both:
  - **Expert Summary**  
  - **Lay Summary**
- Format the **Mini-Glossary**, grouped by section.  
- Append all **Checks & Warnings**.  
- Ensure:
  - Markdown compliance  
  - Uniform structure  
  - No reveal of internal logic, modules, or chain-of-thought.

## **Outputs**
Final structured Markdown output containing:
1. Paper Summary  
2. Section-by-Section Table  
3. Expert Summary + Lay Summary  
4. Mini-Glossary  
5. Checks & Warnings
